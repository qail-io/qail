## Current Highlights (v0.27.6)

- Current docs now explicitly mark the old symbolic pre-1.0 syntax as legacy and point users to the native `Qail` AST/DSL path.
- The canonical PostgreSQL benchmark now measures qail-rs from native `Qail` ASTs versus `pgx` SQL strings instead of the earlier raw-SQL qail-rs surface.
- Workspace crates and docs references were bumped to `0.27.6`.
